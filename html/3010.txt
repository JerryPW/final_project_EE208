// 切换语言功能
function switch_lang()
{
	$(".link-lang").click(function(){
		var lang = $(this).attr('lang');
		if ($.cookie('lang') == lang)
		{
			return false;
		}
		$.cookie('lang', lang);
		location.reload();//刷新当前窗口
		return false;
	});
}

相约国庆校园，寻找初秋美景
时间：2022-09-30 06:36
盛夏已逝，国庆已至，美丽的交大秋景已经来到我们身边。你是否愿意开启一场国庆校园之旅，漫游这熟悉却又有点陌生的校园呢？不如参加上海交通大学校园管理办公室、水源社区、学生联合会一起举办的“相约国庆校园，寻找初秋美景”活动，将你对交大的爱，对秋景的创意定格在镜头里。01 活动信息活动对象：上海交通大学全体在校学生活动时间：2022年10月01日0:00至2022年10月07日20:00主办单位：上海交通大学校园管理办公室主办水源社区站务委员会、上海交通大学学生联合会协办参与方式：参赛用户需注册并登录“水源社区”，在首页“相约国庆校园，寻找深秋美景”活动主题帖下，以回帖方式提交参赛作品。02 参赛要求1. 参赛同学从给定的路线主题中选一个，或者自定路线，绘制“最美校园角落”地图，并提交沿途景点照片。参赛者需要提交作品需包括路线图一张，文字注明路线主题类别；拍摄景点照片至少三张，文字注明景点名称。图片格式需为JPG或PNG。2. 鼓励参赛用户对校园景观进行创意设计，在参赛作品后同时附上设计图和文字说明。（注：优秀创意设计将在未来的校园改造中有所参考~）3. 每位参赛用户活动期间可多次投稿，取一项最高点赞数量的作品进行排名认定。4. 参赛作品必须为本人原创作品，确认具有著作权，不能侵犯任何他人和单位的任何专利权、著作权、商标权及其他合法权益。请选手为自己负责，如有报名作品盗用情况与主办方、协办方无关。03 推荐主题主题一：自然寻踪推荐地点：撷英园、思源湖、仰思坪、大草坪、宇琼坡、捭阖塘、植物园、南苏园等主题二：艺术之旅推荐地点：迸发的力量、雄鹰雕塑、维纳斯雕像、名人类雕塑、植物园里的雕塑、各类小型雕塑等主题三：建筑风貌推荐地点：东晖园、文博楼、东大门、理科楼群、李政道图书馆、思源门等注：如进入相关场馆请遵守场馆防疫要求。更多景点及介绍可登录上海交通大学电子地图，点击右下角“景观”查看。04 活动奖励 人气奖最佳作品奖最具创意奖注：活动评委组由学校相关管理单位教师代表、学生联合会学生代表、水源社区用户代表组成；如遇作品点赞数相同，则发布时间早的作品优先排名；每位参赛用户奖品不可累计，仅发放用户所获最高奖项的对应奖品。05 结语一个转身夏天就成了故事，一个回眸秋天就成了风景。欢迎你在这个假期拍摄下心中的最美校园！感谢“哈啰出行”对本次活动的大力支持！
