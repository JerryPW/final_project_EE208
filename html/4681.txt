// 切换语言功能
function switch_lang()
{
	$(".link-lang").click(function(){
		var lang = $(this).attr('lang');
		if ($.cookie('lang') == lang)
		{
			return false;
		}
		$.cookie('lang', lang);
		location.reload();//刷新当前窗口
		return false;
	});
}

科学统筹协同作战，做好防疫常态化下的防汛防台工作——上海交通大学召开2022年防汛防台工作部署会
时间：2022-07-14 10:12
为有效落实习近平总书记关于安全生产的重要指示精神和上级防汛防台工作要求，进一步提高学校“防大汛、抗大灾、抢大险”的能力，合力建设平安校园。7月14日下午，上海交通大学召开2022年度防汛防台工作部署会。校党委常委、副校长奚立峰出席会议，校防汛防台领导小组各成员单位负责人、各二级单位办公室主任、各物业单位负责人参加了此次会议。会议由校园管理办公室主任王震主持。奚立峰结合今年防汛防台存在的风险点和疫情防控的特殊性，提出三点要求。第一，提高政治站位，筑牢安全底线不松懈。坚持“人民至上、生命至上”，科学统筹疫情防控和防汛防台工作，各部门需利用现阶段的时间将预案落实到位。第二，加强组织领导，形成防汛工作强大合力。要严格落实值班及报告制度，做好信息排摸、报送和处理等，同时做好师生员工防汛防台宣传教育，提高安全防范意识。第三，将工作预案落实落细，发挥预案的防范作用。抓重点、补短板、强弱项。各单位要坚决克服麻痹思想和侥幸心里。夯实防范措施，全力以赴为建设平安校园保驾护航，以实际行动迎接党的二十大胜利召开。会上，王震总结了2021年的防汛防台工作，并根据上级防汛防台工作要求及学校专项工作预案，对2022年疫情防控新形势下的防汛防台工作进行了研判和具体部署。会前，各个校区已陆续开展了防汛防台巡查工作，围绕生活园区、在建工地、物资储备、设施设备等方面及时查漏补缺。通过多部门协同作战，全力保障各校区安全度汛。
