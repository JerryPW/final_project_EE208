// 切换语言功能
function switch_lang()
{
	$(".link-lang").click(function(){
		var lang = $(this).attr('lang');
		if ($.cookie('lang') == lang)
		{
			return false;
		}
		$.cookie('lang', lang);
		location.reload();//刷新当前窗口
		return false;
	});
}

笃行不怠迎盛会 乘势而上向未来——上海交通大学校园管理办公室党支部集体收看二十大开幕会
时间：2022-10-16 15:46
10月16日上午，中国共产党第二十次全国代表大会今天在北京隆重开幕。上海交通大学校园管理办公室党支部组织所有党员收看开幕会直播，认真学习了习近平总书记代表第十九届中央委员会向大会所作的报告。中国共产党第二十次全国代表大会，是在全党全国各族人民迈上全面建设社会主义现代化国家新征程、向第二个百年奋斗目标进军的关键时刻召开的一次十分重要的大会。时长近两小时的报告回顾了党的十八以来砥砺奋进的十年伟大成就，党员们观看直播后，纷纷表示习近平总书记的报告高屋建瓴、心系人民，进一步坚定了理想信念，强化了使命担当，弘扬了伟大建党精神，为未来工作发展指明了方向。党员们表示，新时代砥砺奋进的非凡十年向世人充分展现了习近平新时代中国特色社会主义思想的强大真理力量。习近平特别指出的“实施科教兴国战略，强化现代化建设人才支撑”让人印象深刻。教育、科技、人才是全国建设社会主义现代化国家的基础性、战略性支撑。作为校园管理的工作者，我们应当将“为党育人、为国育才”作为重要的责任和使命，为祖国培养可堪民族复兴大任的时代新人，培养德智体美劳全面发展的社会主义建设者和接班人。在校园管理工作中，坚持以人为本，提升服务师生的管理水平，从探寻城市治理的“绣花针”到实现校园的精美化管理，从了解现代化的科技手段到破解校园师生的生活服务难题，助力构建精准化、智能化、现代化的优质校园。踔厉奋发，乘势而上，我们要坚定不移听党话、跟党走，怀抱梦想又脚踏实地，敢想敢为又善作善成，让青春在全面建设社会主义现代化国家的火热实践中绽放绚丽之花。本次二十大开幕会意义深远，振奋人心，激励广大党员在新征程上担当新使命、展现新作为。校园管理办公室全体党员立足本职岗位、坚定信心、勇毅担当，值此二十大开幕之际，乘势而上、行稳致远，以实实在在的工作成效阔步迈向未来。
