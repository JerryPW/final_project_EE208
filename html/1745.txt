// 切换语言功能
function switch_lang()
{
	$(".link-lang").click(function(){
		var lang = $(this).attr('lang');
		if ($.cookie('lang') == lang)
		{
			return false;
		}
		$.cookie('lang', lang);
		location.reload();//刷新当前窗口
		return false;
	});
}

【公告】2021年上半年交大闵行等校区废旧自行车回收服务招标公告
时间：2021-06-21 16:08
根据工作需要，上海交通大学校园管理办公室计划面向社会公开招标，对闵行、徐汇校区废旧自行车物资提供回收服务，诚邀符合条件的供应商投标，并于2021年6月24日前报名反馈投标意向，2021年6月25日前递交投标文件。学校将根据投标公司资质、经验、服务、报价等综合评审确定中标人。相关采购需求和要求如下： 一、服务内容1、回收我校闵行校区S4高速公路下堆放的废旧自行车，总计数量约为600辆，实际工作量以各家公司现场自行踏勘为准，总量包干。2、回收我校徐汇长宁校区废旧自行车，总计数量约为100辆，负责完成清运等工作。总计：700辆。 二、服务要求1、员工服务态度良好，素质高，配合度好，安全意识强。2、响应速度较快，遇紧急情况时，可提供快速优质的服务。3、有企事业单位废旧物资回收经验、依法依规进行自行车处置工作。4、全权负责回收过程中的员工安全。 三、基本条件1、中华人民共和国境内注册，具备工商行政管理部门核发的企业法人营业执照，具有独立法人资质，进行工商税务登记且年检合格，营业执照在有效期内。2、具备再生资源回收经营资质，生产性废旧金属收购资质，主营范围含废旧物资回收。3、近三年内，在经营活动中没有重大违法行为，同类项目服务中没有发生过重大安全事故。 四、投标文件1、有效期内的营业执照副本复印件。2、法定代表人身份证复印件，或法人代表授权书及授权代表身份证复印件。3、相关资质证书复印件。4、公司简介，含公司规模、项目经验、人员情况、运输和作业设备配置、处置所需时限、服务费用等。以上所有材料需一式伍份。 五、评标办法（一）评标依据和原则1、本评标办法作为本项目择优选定中标人的依据，在评标全过程中应遵照执行。2、评委共5名校内专家。3、各评委根据材料进行打分、排序，并填写评分依据。4、 评审委员会根据各评委推荐意见，汇总后排名第1位的投标人推荐为中标人。（二）评标细则1、各评委根据以下几方面进行打分，总分100分（企业资质20分、相关经验20分、项目报价60分）。2、各评委根据打分高低进行排序；评审委员会对各单位顺序号进行累加，按照升序排列，累加数量最小的单位排名第1位，推荐为中标人；3、如第一轮评审中，排序第1名出现并列，则评审委员会进行第二轮投票（规则同第一轮）推荐中标人。 六、联系方式单位：上海交通大学校园管理办公室地址：上海交通大学闵行校区，东川路800号，基建楼108室联 系 人：王老师联系电话：021-54743939 校园管理办公室 2021年6月21日
