// 切换语言功能
function switch_lang()
{
	$(".link-lang").click(function(){
		var lang = $(this).attr('lang');
		if ($.cookie('lang') == lang)
		{
			return false;
		}
		$.cookie('lang', lang);
		location.reload();//刷新当前窗口
		return false;
	});
}

砥砺奋进正当时，勇毅前行新征程——校园管理办公室党支部开展七月学习交流会
时间：2022-07-21 06:08
为了深入学习贯彻“上海市第十二次党代会精神”，助力推进学校管理服务高质量发展，2022年7月15日上午，校园管理办公室党支部举行了7月学习交流会。会议由党支部书记王震主持，全体党员参加。党支部书记王震带领大家学习了市委书记李强关于《弘扬伟大建党精神 践行人民城市理念 加快建设具有世界影响力的社会主义现代化国际大都市》的报告，回顾总结过去五年的工作、未来五年的建设目标、开创未来发展新局面和党的建设四个部分，并针对校园管理工作相关的部分进行了重点学习分享。王震强调，要将上海城市治理的优秀做法运用到校园管理工作中去，具备绣花般的细心、耐心和卓越心，助力建设更具魅力、更富有人文情怀的校园。支部全体党员集体学习了2022年高校党组织示范微党课展播第6讲和《新时代 新奇迹·2017-2022”上海发展成就展》相关视频文件，并就暑期期间的工作规划作交流分享。上海市第十二次党代会客观总结了牢记嘱托、砥砺奋进的五年，科学谋划了未来发展的新局面。通过此次学习交流会，支部党员认真研读，深刻领会，针对校园管理相关的人民城市治理理念和先进方法做到学深悟透，充分与本职工作结合起来，持续提升校园管理服务能力，以实际行动迎接党的二十大胜利召开。
